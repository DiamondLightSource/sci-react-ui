# Theme colours

Theme colours provide the complete semantic colour palette available in DiamondDS. Each colour role is shown in both light and dark modes using the design system tokens. These values are intended as a reference for implementation and should be used through their semantic names rather than by selecting colours based on appearance.

<div className="ds-note">

This page is a token reference. If you're deciding which colour role to use, read the **[Colour](/docs/foundations-colour--docs)** guidance first.

</div>
