# Diamond Design System ❖

The design system for Diamond Light Source, providing a shared foundation for building clear, consistent and reliable scientific interfaces.

It evolves SciReactUI into a semantic, scalable system for data acquisition, analysis and operational tools across Diamond.

Built on structure and precision, the system helps teams create interfaces that behave predictably across scientific workflows, applications and light/dark modes.

## What it helps us do

<ul>
  <li>Create clearer, more predictable scientific interfaces.</li>
  <li>Bring consistency across tools and beamlines.</li>
  <li>Reduce cognitive load in complex workflows.</li>
  <li>Speed up development with reusable components.</li>
  <li>Ensure new tools start coherent and stay aligned.</li>
</ul>
